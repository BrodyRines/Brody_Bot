#Discord Bot Project

Personal Discord Bot I made usinf python, and the Discord API

##Summary

I am actively adding and updating this Discord bot, but as it stands, it can message directly to users, play Rock, Paper, Scissors, give ideas to do when your bored, and has a few different bot events that can send messages or interact with users. I built it on VScode and used the discord python library to make it, and used the Discord API to import the bot into the server, and interact and use features on the Discord app. I built this app to get more comfortable with using different API's, and made it as a personal project to test my skills and use it with my friends.

###Dependencies

The dependencies are in the requirements.txt, and I will go more into set up in the 'Installing' and 'Executing Program' sections

###Installing

To install the libraries needed for this program, go into the terminal and type 'pip install -r .\requirements.txt'. This will automatically download the libraries used for this program that are not apart of Pythons original library. You will also need to install Discord onto your computer to be able to access the bot, and to import it in from your IDE.

###Executing Program

The next important thing you must do is to get a Discord API key, which is accessed by signing into your account on the Discord Developer Portal on your browser. Once you aquire your key, you must make a .env file, and store your key in it assigned to the variable 'DISCORD_TOKEN'. You must then create a new bot on Discord Developer Portal and give it most of the permissions, mainly for sending messages and interacting with the server and users. You can then import the bot from their into the server of your choosing.
